# footbot

Run
---
    $ python -m footbot

Access with credentials
-----------------------

    $ export FPL_LOGIN=my@email.com
    $ export FPL_PASSWORD=mypassword
    $ ./curl.sh "<footbot url>/optimise_team/5002301?start_event=5"

Test
----
    $ python -m pytest

Format
------
    $ ./format.sh
